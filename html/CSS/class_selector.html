Class Selector

Define class selector?

    Applying the styling or css properties by using the class attribute in the css file is known as class selector.
    Ex: <p class="paragraph-class"> This is a paragraph tag </p>
    
    .paragraph-class {
        color: blue;
        text-align: center;
        text-decoration: underline;
        border: dashed;
        border-color: brown;
    }
    In th above example clearly says, we applied the styling by using the class value.
