ID Selector

Define an id selector?

    Applying the styling or css properties using the id attribute in css file is known as ID selector.

    Ex: <p id="p1-tag"> Hi welcome to id selector. </p>
    
    #p1-tag {
        color: green;
    }

    ==>In the above example, while styling we are using the id value to style for that particular id related element.
    Note: id's most of the time unique.
