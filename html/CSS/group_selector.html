Define class selector?

    Applying the styling or css properties by using the tags of multiple contents at the same time in the css file is known as class selector.
    Ex: <p> This is a paragraph tag </p>
        <h1> This is a paragraph tag </h1>
    
    p,h1 {
        color: blue;
        text-align: center;
        text-decoration: underline;
        border: dashed;
        border-color: brown;
    }
    
    In th above example clearly says, we applied the styling by using the tag of multiple contents.
