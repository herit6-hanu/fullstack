Define class selector?

    Applying the styling or css properties by using the * symbol in the css file, here * refers that we are applying the contents to all the elements which are inside the HTML file is known as universe_selector.
    Ex: <p> This is a paragraph tag </p>
        <h1> This is a paragraph tag </h1>
    
    * {
        color: blue;
        text-align: center;
    }
    
    In th above example clearly says, we applied the styling by using the * means that will apply the styling for all the contents in the html file.
